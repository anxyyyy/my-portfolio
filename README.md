# Personal Portfolio Website 🚀

Modern, clean, and highly interactive frontend developer portfolio with a premium dark theme and responsive layout.

🔗 **Live Demo:** [ВСТАВЬ_СЮДА_ССЫЛКУ_НА_GITHUB_PAGES]

## ✨ Features

- **Premium Dark Design**: Styled with a curated palette, glassmorphism headers, and smooth hover micro-animations.
- **3D Tilt Badge**: An interactive developer ID card on the Hero section that reacts dynamically to mouse movement using pure JS mathematics.
- **Dynamic Portfolio Navigation**: Interactive tab switcher showing Projects, Certificates, and Tech Stack.
- **Detailed Project Modals**: Clickable project cards that open details overlay.
- **Fullscreen Lightbox**: Pop-up preview for training certificates.
- **Guestbook (Comments)**: Fully functional review board utilizing HTML5 File Reader (for image uploads) and persisting comments in `localStorage`.
- **Responsive Layout**: Mobile-first design adapting perfectly from desktop screens to tablets and smartphones.

## 🛠️ Built With

- **HTML5**: Semantic markup for layout.
- **Tailwind CSS**: Utility-first CSS framework for layout, grids, and adaptive styling.
- **Vanilla JavaScript**: Pure JS logic for navigation, 3D tilt, modal windows, and localStorage state management.
- **Lucide Icons**: Modern vector icon pack.
